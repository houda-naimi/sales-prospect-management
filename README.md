# prospect-management-system

This project is a mobile management system designed for handling prospects, prospecting operations, and after-sales services through the integration of a complaint management module dedicated to sales agents.

The application enables sales agents to efficiently manage their daily activities directly from a mobile device.

---

#  Technologies Used

## Mobile Front-End
- Xamarin.Forms
- C#
- XAML

## Back-End
- ASP.NET Web API
- Entity Framework
- SQL Server

## Architecture
- MVVM Architecture
- REST API

---

#  Main Features

## Authentication

The system:
- Verifies the purchased license.
- Grants access using a username and password.

## Prospect Management

Sales agents can:

- View the history of phone calls.
- Add new prospects.
- Edit prospect information.
- View prospect details.
- Contact prospects:
  - By email
  - By message
  - By phone call
- View the complete list of prospects.

## Prospecting Management

The application allows users to:

- Add prospecting records.
- Edit existing prospecting records.
- Check the availability of real estate properties:
  - Apartments
  - Houses
  - Commercial premises
  - Land plots

## Appointment Management

Available features:

- Add appointments
- View appointments
- Edit appointments
- Delete appointments

## Complaint Management

The after-sales service module allows users to:

- Add complaints
- View complaints
- Edit complaints
- Delete complaints

## Payment Deadline Management

Sales agents can:

- View unpaid invoices sorted by due date.
- Search unpaid invoices by:
  - Client name
  - Amount
  - Due date
- Contact clients via email with payment deadline details.
